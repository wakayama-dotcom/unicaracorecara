# これから。フローツール

理想と現実のギャップを見える化し、4つの選択肢を根拠つきで提示する一括ツールです。

## Chatworkで他社に共有する場合

- **`standalone.html`** を利用してください（CSS・JSを内蔵した1ファイルです）。
- Netlify Drop や GitHub Pages などで `standalone.html` を公開し、発行された **URLをChatworkで送る**と、相手がリンクをクリックするだけでツールを開けます。
- 詳しくは **`CHATWORK共有手順.md`** を参照してください。

## 流れ

1. **現在地を整理する** — 6領域（収入・働き方・職場環境・仕事内容・スキル・生活）で「理想」と「現実」を入力
2. **ギャップを確認する** — 理想と現実の差を一覧で確認し、各領域のギャップの大きさ（小・中・大）を選択
3. **選択肢を見る** — ①転職+副業／②転職／③副業／④現状の見直し を比較カードで表示（各カードに根拠つき）

## 使い方

- `index.html` をブラウザで開く（ローカルでそのまま動作します）
- または簡易サーバーで配信する場合:  
  `npx serve .` や `python3 -m http.server 8000` で `tool` フォルダをルートにしてアクセス

## ファイル

- `index.html` — ページ構成・ステップインジケーター・3パネル
- `styles.css` — レイアウト・見た目
- `app.js` — フォーム描画・データ収集・ギャップ表示・選択肢カードと根拠の生成

設計の詳細はプロジェクト直下の `docs/` を参照してください。
