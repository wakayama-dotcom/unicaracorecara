# Unicara「これから。」サービス概要

> このままでいいのか。を最初に相談する場所

---

## 1. サービスコンセプトと存在意義

「これから。」は、単なる求人紹介を行うエージェントではありません。人生の岐路に立つ相談者が、自らの意志で納得して道を選べるよう伴走する**「意思決定支援」**の専門機関です。

- **キャッチコピー：** 「このままでいいのか。」を最初に相談する場所
- **サービスの本質：** 転職先という「場所」の提供ではなく、自分で確信して選べる「状態そのもの」を提供すること
- **メッセージ：** キャリアへの不安や迷いは人生に真剣に向き合っている証。一人で抱え込まず、対話を通じて本音を整理することで、進むべき道が見えてくる

---

## 2. MVV（ミッション・ビジョン・バリュー）

|  | 内容 |
|---|------|
| **Vision** | キャリアを考え続けられる社会を |
| **Mission** | これからの迷いを確信に変える |
| **Core Value 1** | 現在地を整理し、キャリアの意思決定に伴走する（対話の質を重視。転職すべきかどうかを自分で納得して判断できるまで寄り添う） |
| **Core Value 2** | 転職だけではない選択肢で、人生を設計する（FP事業・副業支援のノウハウを活用し、転職・副業・現職改善などトータルでサポート） |

---

## 3. 市場背景とポジショニング

- **ターゲット年代：** 25～34歳（転職市場のボリュームゾーン、20・30代合計で市場の63.7%）
- **ターゲット職種：** 営業、販売、サービス職をメインに支援
- **特化フェーズ：** Phase 0〜2（なんとなくモヤモヤ／動けない／転職すべきか本気で迷っている潜在層）。Phase 3（転職決意・条件整理済み）は大手エージェントが注力する領域
- **構造的課題の解決：** 「孤独な意思決定」によるリスクへの恐怖や判断材料の不足を、対話によって解消

---

## 4. サービス工程（全9工程）

| 工程 | カテゴリ | 内容・目的 | 強み・特徴 |
|------|----------|------------|------------|
| 01 | サービス紹介 | 転職ありきではない支援方針・市場構造の説明 | 急かされず、何を変えたいか考える場だと認識 |
| 02 | 現在地整理 | 過去の判断軸と現在の悩みを「6領域」でヒアリング | 自己分析を一人で終わらせず、迷いを言語化 |
| 03 | 人生設計 | 理想の未来から逆算し、転職・副業・継続を検討 | 顧客相談満足度100%。理想の人生把握マップ |
| 04 | 判断軸の確立 | 絶対条件/妥協条件/NG条件の整理、優先順位の決定 | 自分の言葉で「なぜその会社か」を説明できる |
| 05 | 書類作成 | 履歴書・職務経歴書の添削および推薦文作成 | **書類通過率90%超**。経歴書にない魅力を抽出 |
| 06 | 企業・求人紹介 | 軸に合う企業を提案。入社後のリアルな情報共有 | **約11万社**の求人から、迷わず意思決定できる |
| 07 | 面接対策 | 経験・意思の言語化、模擬面接の実施 | **PREP法**を活用した練習を複数回実施 |
| 08 | 意思決定・不安解消 | 内定前後の悩み吸い上げ、比較整理の再確認 | 第三者の視点により、確信を持って決断できる |
| 09 | 入社後フォロー | 3ヶ月/6ヶ月/1年の定期面談 | キャリアの「かかりつけ医」としての継続支援 |

## 5. 6領域フレームワーク（人生設計の共通言語）

1. **収入** — 給与、成果報酬、将来の年収推移
2. **働き方** — 残業時間、リモート、副業の可否、自律性
3. **職場環境** — 人間関係、評価制度、企業文化（MVV）
4. **仕事内容** — 業務の性質、適性、やりがい、裁量
5. **スキル** — 市場価値、専門性、ポータブルスキル
6. **生活** — 家族との時間、健康、理想のライフスタイル

---

## 6. 支援インフラと体制

- **プラットフォーム：** 大手も使用するエージェントサービス「circus」を導入、全国約110,000件の求人
- **UGS（Unicara Group Service）連携：**
  - **財務視点：** FP・保険事業によるライフプランシミュレーション（41社の保険対応）
  - **副業ノウハウ：** 副業支援の実績を活用した、転職だけに頼らないキャリア形成
- **ボードメンバー：**
  - **宮崎 紗佑里** — 女性視点での本音の深掘り、感情に寄り添う支援
  - **池田 和也** — M&A仲介経験、年間300社超の経営者面談に基づく企業分析能力

---

## 7. 顧客が得られるベネフィット

- **変化の明確化** — 「転職したいか」ではなく「何を変えたいか」が明確になる
- **理想の具体化** — 将来どうなりたいかの理想が可視化される
- **選択肢の拡張** — 転職以外の解決策（副業・現職改善）を知ることができる
- **確信ある行動** — 自分の選択に自信を持ち、迷いなく一歩を踏み出せる
